# 权限分享
你可以通过接口显示或隐藏权限分享弹窗。
## 示例
```js
myComponent.invoke.share(true, {
  top: '50px',
  right: '10px',
}).then(function(response) {
  const { code, msg } = response;
  // ...
});
```

## 参数
|属性|	类型|	是否必须	|说明|
| ---|----- | -------|------ |
|visible|	Boolean	|否|	是否显示|
|style| Object|否|分享弹窗样式|

## 返回
|属性|	类型|	说明|
| ---|----- | ------- |
|code|	Number |	0：成功 、 -1：失败 |
|msg|	String |	信息 |
