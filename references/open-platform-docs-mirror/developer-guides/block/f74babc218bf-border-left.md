# border-left

## 介绍

用于设置左边框的样式，该属性可按顺序设置`border-left-width`、`border-left-style`、`border-left-color`。

## 语法

```css
/* 左边框是粗双线红边框 */

border-left: thick double red;

/* 左边框是细点线蓝边框 */

border-left: thin dotted blue;

/* 左边框是10px虚线橙边框 */

border-left: 10px dashed orange;
```

## 标准化语法

```css
border-left: <line-width> || <line-style> || <color>
```
