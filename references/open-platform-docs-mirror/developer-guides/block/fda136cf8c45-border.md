# border

## 介绍

用于设置边框的样式，该属性可按顺序设置`border-width`、`border-style`、`border-color`。

## 语法

```css
/* 四个边框是粗双线红边框 */

border: thick double red;

/* 四个边框是细点线蓝边框 */

border: thin dotted blue;

/* 四个边框是10px虚线橙边框 */

border: 10px dashed orange;
```

## 标准化语法

```css
border: <line-width> || <line-style> || <color>
```
