# 部门接口必填校验逻辑升级

## 概述
部门新建接口存在字段必填校验与前端页面不一致的问题，会导致部分场景下业务配置的必填字段写入为空，因此需要对这种问题进行修复。

## 生效时间

本次不兼容变更预计于 2026 年 3 月 19 日起逐步生效，具体灰度与全量节奏将以开放平台实际发布为准。

## 变更详情

- **部门接口必填校验逻辑升级**
	- **涉及接口：**[创建部门](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/corehr-v1/department/create)
	- **更新内容：** 修复了接口字段必填校验与前端页面不一致的问题。后续所有字段必填校验将完全对齐业务在【飞书人事-设置-组织配置-部门字段配置】设置的字段必填属性

## 变更影响
修复后，接口将严格校验【飞书人事-设置-组织配置-部门字段配置】页面中设置为“必填”的字段。若接口请求中未传入这些必填字段的值，将被拦截并返回报错。

## 开发者建议

请检查现有调用逻辑，确保已传入所有业务配置的必填字段，避免接口调用失败.

如需适配协助，请点击右下角按钮，联系开放平台技术支持。
