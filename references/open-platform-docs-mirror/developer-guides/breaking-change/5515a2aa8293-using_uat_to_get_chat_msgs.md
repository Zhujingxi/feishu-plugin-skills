# 开启对外共享能力的应用支持使用 user_access_token 获取外部会话中的消息

### 变更事项

为了提供更好的应用对外共享能力，**2026 年 4 月 29 日起，开启对外共享能力的应用使用本租户内成员的 `user_access_token` 获取外部会话（包括外部单聊与群聊）中的消息时，可以正常获取，将不再返回错误码。** 各受影响接口的变更内容如下：

**1. [获取会话历史消息](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/message/list)**
- **变更前**：开启对外共享能力的应用**不支持**使用 `user_access_token` 获取任何会话（包括内部、外部的单聊与群聊）中的消息，返回错误码。
- **变更后**：开启对外共享能力的应用**支持**使用 `user_access_token` 获取该用户所在会话（包括内部、外部的单聊与群聊）中的消息。

**2. [获取指定消息的内容](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/message/get)**
- **变更前**：开启对外共享能力的应用**不支持**使用 `user_access_token` 获取外部会话（包括外部单聊与群聊）中的消息，返回错误码。
- **变更后**：开启对外共享能力的应用**支持**使用 `user_access_token` 获取该用户所在外部会话（包括外部单聊与群聊）中的消息。

> **是否跟随客户端版本**：不跟版<br>
> **预计生效时间**：2026-04-29<br>

### 潜在影响

本次变更**预计不会对现有应用造成阻塞性影响**，但需关注应用获取到的消息可能包含**外部租户**参与的会话内容。

### 解决方案

无。
