# VideoContext.seek(number position)

跳转到指定位置

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | V4.3.0+ | V4.3.0+ | V6.1.0+ | V7.35.0+ | 预览
网页应用 | **X** | **X** | **X** | **X** | /

## 输入

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
position | number | 是 | &nbsp; | 视频位置，单位秒(s)<br>**示例值**：20

## 输出
无

## 示例代码

```html
<video id="myVideo"
  src="http://sf3-cn.feishucdn.com/obj/ttfe/tma/test.mp4"
  loop="{{true}}"
  initial-time=5
  auto-fullscreen="{{true}}"
  bindtimeupdate="bindupdatetime"
  muted="{{myMuted}}">
</video>
```

```js
Page({
  onReady (res) {
    this.videoContext = tt.createVideoContext('myVideo');
    this.videoContext.play();
    this.VideoContext.seek(20);
  }
})
```
