# Animation.left(number|string value)

设置 left 值

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **✓** | V7.35.0+ | 预览
网页应用 | **X** | **X** | **X** | **X** | /

## 输入

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
value | number | string | 是 | / | 长度值，如果传入 number 则默认使用 px，可传入其他自定义单位的长度值

## 输出

返回值：

`Animation` 实例

## 示例代码

```js
const animation = tt.createAnimation();

animation.left(20).step();
```
