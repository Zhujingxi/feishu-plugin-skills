# Animation.translateZ(number translation)
对 Z 轴平移

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **✓** | V7.35.0+ | 预览
网页应用 | **X** | **X** | **X** | **X** | /

## 输入

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
translation | number | 是 | / | 在 Z 轴平移的距离，单位为 px

## 输出

返回值：

`Animation` 实例

## 示例代码

```js
  const animation = tt.createAnimation();

animation.translateZ(10).step();
  ```
