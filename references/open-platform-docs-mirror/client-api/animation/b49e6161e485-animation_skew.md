# Animation.skew(number ax, number ay)
对 X、Y 轴坐标进行倾斜

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **✓** | V7.35.0+ | 预览
网页应用 | **X** | **X** | **X** | **X** | /

## 输入

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
ax | number | 是 | / | 对 X 轴坐标倾斜的角度，范围 [-180, 180]
ay | number | 是 | / | 对 Y 轴坐标倾斜的角度，范围 [-180, 180]

## 输出

返回值：

`Animation` 实例

## 示例代码

```js
  const animation = tt.createAnimation();

animation.skew(90, 90).step();
  ```
