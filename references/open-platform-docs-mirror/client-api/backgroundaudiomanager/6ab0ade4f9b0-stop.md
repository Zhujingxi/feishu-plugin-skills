# BackgroundAudioManager.stop()

背景音频停止播放

## 支持说明

应用能力 | Android | iOS | PC | 预览效果
---|---|---|---|---
小程序 | V5.20.0+ | V5.20.0+ | **X** | 预览
网页应用 | **X** | **X** | **X** | 预览

## 输入
无

## 输出
无
## 示例代码

```js
const bam = this.backgroundAudioManager = tt.getBackgroundAudioManager();
bam.src = 'https://someaudiourl';
bam.play();
bam.stop();
```
