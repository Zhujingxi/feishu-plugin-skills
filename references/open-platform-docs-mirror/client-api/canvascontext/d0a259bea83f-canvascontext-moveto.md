# CanvasContext.moveTo(number x, number y)

移动绘制点

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **✓** | V7.35.0+ | 预览
网页应用 | **X** | **X** | **X** | **X** | /

## 输入

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
x | number | 是 | &nbsp; | x 坐标
y | number | 是 | &nbsp; | y 坐标

## 输出

无

## 示例代码

```javascript
const ctx = tt.createCanvasContext(canvasId);

ctx.moveTo(10, 50);
ctx.lineTo(100, 50);
ctx.stroke();

ctx.draw();
```
