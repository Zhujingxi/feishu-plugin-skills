# RecorderManager.pause()

暂停录音

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **X** | V7.43.0+ | 预览
网页应用 | **✓** | **✓** | **X** | V7.43.0+ | 预览

## 输入

无

## 输出
无

## 示例代码

```js
const recorderManager = tt.getRecorderManager();
const options = {
  duration: 100000,
  sampleRate: 44100,
  numberOfChannels: 2,
  encodeBitRate: 320000,
  frameSize: 50
};

recorderManager.start(options);
recorderManager.pause();
```
