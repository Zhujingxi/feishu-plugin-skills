# NFCAdapter.offDiscovered(function callback)

取消监听 NFC Tag

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | V3.38.0+ | **X** | **X** | V7.35.0+ | 预览
网页应用 | V3.44.0+ | **X** | **X** | V7.35.0+ | 预览

## 输入

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
callback | function | 是 | &nbsp; | 该事件的回调函数

## 输出
无

## 示例代码

**注意事项**：这里仅示例当前 API 使用，实际流程请参考：[接入 NFC](https://open.feishu.cn/document/uYjL24iN/ugTN4YjL4UDO24CO1gjN)

```js
var adapter = tt.getNFCAdapter();
var onDiscoveredCallback = (res) => {
   console.log(JSON.stringify(res));
};
adapter.onDiscovered(onDiscoveredCallback);
adapter.offDiscovered(onDiscoveredCallback);
```

`fail`返回对象中会包含[errno属性](https://open.feishu.cn/document/uYjL24iN/uAjMuAjMuAjM/errno)，代表错误码。
