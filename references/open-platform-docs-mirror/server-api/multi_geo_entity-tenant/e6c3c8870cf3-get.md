# 获取数据驻留地理位置列表

获取租户可用的数据驻留地理位置列表

## 请求

基本 | &nbsp;
---|---
HTTP URL | https://open.feishu.cn/open-apis/security_and_compliance/v1/multi_geo_entity/tenant
HTTP Method | GET
接口频率限制 | [100 次/分钟](https://open.feishu.cn/document/ukTMukTMukTM/uUzN04SN3QjL1cDN)
支持的应用类型 | Custom App
权限要求<br>**调用该 API 所需的权限。开启其中任意一项权限即可调用**<br>开启任一权限即可 | 查看数据驻留租户信息(security_and_compliance:multi_geo_entity.tenant:readonly)<br>查询、更新员工的数据驻留地(security_and_compliance:user_migration:multi-geo)

### 请求头

名称 | 类型 | 必填 | 描述
---|---|---|---
Authorization | string | 是 | `tenant_access_token`<br>或<br>`user_access_token`<br>**值格式**："Bearer `access_token`"<br>**示例值**："Bearer u-7f1bcd13fc57d46bac21793a18e560"<br>[了解更多：如何选择与获取 access token](https://open.feishu.cn/document/uAjLw4CM/ugTN1YjL4UTN24CO1UjN/trouble-shooting/how-to-choose-which-type-of-token-to-use)

## 响应

### 响应体

名称 | 类型 | 描述
---|---|---
code | int | 错误码，非 0 表示失败
msg | string | 错误描述
data | \- | \-
tenant | tenant | 数据驻留租户信息
available_geo_locations | string\[\] | 可选地理位置列表

### 响应体示例
```json
{
    "code": 0,
    "msg": "success",
    "data": {
        "tenant": {
            "available_geo_locations": [
                "sg"
            ]
        }
    }
}
```
