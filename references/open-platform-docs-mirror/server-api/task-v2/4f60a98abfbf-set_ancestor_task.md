# 设置父任务

管理任务的父子关系，支持设置任务的父任务，或者将任务转为独立任务。

## 请求

基本 | &nbsp;
---|---
HTTP URL | https://open.feishu.cn/open-apis/task/v2/tasks/:task_guid/set_ancestor_task
HTTP Method | POST
接口频率限制 | [1000 次/分钟、50 次/秒](https://open.feishu.cn/document/ukTMukTMukTM/uUzN04SN3QjL1cDN)
支持的应用类型 | Custom App、Store App
权限要求<br>**调用该 API 所需的权限。开启其中任意一项权限即可调用** | 查看、创建、更新、删除任务(task:task:write)

### 请求头

名称 | 类型 | 必填 | 描述
---|---|---|---
Authorization | string | 是 | `tenant_access_token`<br>或<br>`user_access_token`<br>**值格式**："Bearer `access_token`"<br>**示例值**："Bearer u-7f1bcd13fc57d46bac21793a18e560"<br>[了解更多：如何选择与获取 access token](https://open.feishu.cn/document/uAjLw4CM/ugTN1YjL4UTN24CO1UjN/trouble-shooting/how-to-choose-which-type-of-token-to-use)
Content-Type | string | 是 | **固定值**："application/json; charset=utf-8"

### 路径参数

名称 | 类型 | 描述
---|---|---
task_guid | string | 任务的GUID，可从任务详情字段获取<br>**示例值**："e297ddff-06ca-4166-b917-4ce57cd3a7a0"

### 请求体

名称 | 类型 | 必填 | 描述
---|---|---|---
ancestor_guid | string | 否 | 父任务的guid，可从任务详情字段获取， 未设置时作用为转独立任务<br>**示例值**："e297ddff-06ca-4166-b917-4ce57cd3a7a0"

### 请求体示例
```json
{
    "ancestor_guid": "e297ddff-06ca-4166-b917-4ce57cd3a7a0"
}
```
