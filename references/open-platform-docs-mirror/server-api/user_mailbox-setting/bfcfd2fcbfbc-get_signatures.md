# 获取用户的签名列表

获取用户的签名列表

## 请求

基本 | &nbsp;
---|---
HTTP URL | https://open.feishu.cn/open-apis/mail/v1/user_mailboxes/:user_mailbox_id/settings/signatures
HTTP Method | GET
接口频率限制 | [5 次/秒](https://open.feishu.cn/document/ukTMukTMukTM/uUzN04SN3QjL1cDN)
支持的应用类型 | Custom App
权限要求<br>**调用该 API 所需的权限。开启其中任意一项权限即可调用** | 查询用户的企业邮箱(mail:user_mailbox:readonly)

### 请求头

名称 | 类型 | 必填 | 描述
---|---|---|---
Authorization | string | 是 | `user_access_token`<br>**值格式**："Bearer `access_token`"<br>**示例值**："Bearer u-7f1bcd13fc57d46bac21793a18e560"<br>[了解更多：如何选择与获取 access token](https://open.feishu.cn/document/uAjLw4CM/ugTN1YjL4UTN24CO1UjN/trouble-shooting/how-to-choose-which-type-of-token-to-use)

### 路径参数

名称 | 类型 | 描述
---|---|---
user_mailbox_id | string | 用户邮箱地址，作为用户邮箱身份标识。使用 user_access_token 调用时，可使用占位符 `me` 表示当前授权用户的主邮箱。<br>**示例值**："user@example.com"

## 响应

### 响应体

名称 | 类型 | 描述
---|---|---
code | int | 错误码，非 0 表示失败
msg | string | 错误描述
data | \- | \-
signatures | user_mailbox_signature\[\] | 用户邮箱签名列表
id | string | 签名 ID
name | string | 签名名称
content | string | 签名内容（HTML 格式）
signature_type | string | 签名类型，可选值：USER（用户签名）、TENANT（租户签名）
signature_device | string | 签名适用设备类型，可选值：PC、MOBILE
template_json_keys | string\[\] | 企业签名模板变量渲染
images | user_mailbox_siganature_image\[\] | 签名最后更新时间戳
image_name | string | 签名图片名称
file_key | string | 签名图片的文件 key，用于上传和下载图片
cid | string | 签名图片的 Content-ID，用于在邮件中内嵌引用
file_size | string | 签名图片文件大小，单位为字节
image_width | int | 签名图片宽度，单位为像素
image_height | int | 签名图片高度，单位为像素
download_url | string | 图片下载url
user_fields | map&lt;string, user_mailbox_signature_i18n_val&gt; | 企业签名模版变量值
usages | user_mailbox_signature_usage\[\] | 用户邮箱签名使用情况列表
email_address | string | 邮箱地址
send_mail_signature_id | string | 发送邮件时使用的签名 ID
reply_signature_id | string | 回复邮件时使用的签名 ID
