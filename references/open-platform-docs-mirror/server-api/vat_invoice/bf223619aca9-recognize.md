# 识别文件中的增值税发票

增值税发票识别接口，支持JPG/JPEG/PNG/PDF/BMP/OFD六种文件类型的一次性的识别。
文件大小需要小于10M。
**注意事项**：单租户限流：10QPS，同租户下的应用没有限流，共享本租户的 10QPS 限流

HTTP Method | POST
接口频率限制 | [10 次/秒](https://open.feishu.cn/document/ukTMukTMukTM/uUzN04SN3QjL1cDN)
支持的应用类型 | Custom App、Store App
权限要求<br>**调用该 API 所需的权限。开启其中任意一项权限即可调用** | 增值税发票识别(document_ai:vat_invoice:recognize)

## 响应

### 响应体

名称 | 类型 | 描述
---|---|---
code | int | 错误码，非 0 表示失败
msg | string | 错误描述
data | \- | \-
vat_invoices | vat_invoice\[\] | 增值税发票信息
entities | vat_entity\[\] | 识别出的实体列表
type | string | 识别的实体类型<br>**可选值有**：<br>- invoice_name：发票抬头<br>- invoice_code：发票代码<br>- invoice_no：发票号码<br>- invoice_date：开票日期<br>- total_price：合计金额（不含税）<br>- total_tax：合计税额<br>- big_total_price_and_tax：合计总额（大写）<br>- check_code：校验码<br>- total_price_and_tax：合计总额<br>- buyer_name：购买方名称<br>- buyer_taxpayer_no：购买方纳税人识别号<br>- buyer_address_phone：购买方地址&电话所有人<br>- buyer_account：购买方开户行&账号<br>- seller_name：销售方名称<br>- seller_taxpayer_no：销售方纳税人识别号<br>- seller_address_phone：销售方地址&电话<br>- seller_account：销售方开户行&账号<br>- payee：收款人<br>- invoice_date：开票日期<br>- password_area：密码区<br>- remarks：备注<br>- reviewer：复核人<br>- drawer：开票人<br>- is_sealed：是否盖章<br>- seller_name_in_seal：印章内销售方名称<br>- seller_taxpayer_no_in_seal：印章内销售方纳税人识别号<br>- invoice_special_seal：印章名称<br>- machine_num：机器编号
value | string | 识别出字段的文本信息
items | kv_entity\[\]\[\] | 识别出的票据详细信息
