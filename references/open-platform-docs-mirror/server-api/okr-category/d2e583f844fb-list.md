# 获取所有 OKR 分类

获取系统中所有可用的 OKR 分类的分页列表。每个分类包括 ID、名称（多语言）、颜色、类型、启用状态以及创建/更新时间戳等元数据。

## 请求

基本 | &nbsp;
---|---
HTTP URL | https://open.feishu.cn/open-apis/okr/v2/categories
HTTP Method | GET
接口频率限制 | [100 次/分钟](https://open.feishu.cn/document/ukTMukTMukTM/uUzN04SN3QjL1cDN)
支持的应用类型 | Custom App
权限要求<br>**调用该 API 所需的权限。开启其中任意一项权限即可调用** | 查看 OKR 设置(okr:okr.setting:read)

### 请求头

名称 | 类型 | 必填 | 描述
---|---|---|---
Authorization | string | 是 | `tenant_access_token`<br>或<br>`user_access_token`<br>**值格式**："Bearer `access_token`"<br>**示例值**："Bearer u-7f1bcd13fc57d46bac21793a18e560"<br>[了解更多：如何选择与获取 access token](https://open.feishu.cn/document/uAjLw4CM/ugTN1YjL4UTN24CO1UjN/trouble-shooting/how-to-choose-which-type-of-token-to-use)

### 查询参数

名称 | 类型 | 必填 | 描述
---|---|---|---
page_size | int | 否 | 分页大小，最大 100<br>**示例值**：10<br>**默认值**：`10`<br>**数据校验规则**：<br>- 取值范围：`1` ～ `100`
page_token | string | 否 | 分页标记，第一次请求不填，表示从头开始遍历；分页查询结果还有更多项时会同时返回新的 page_token，下次遍历可采用该 page_token 获取查询结果<br>**示例值**：eVQrYzJBNDNONlk4VFZBZVlSdzlKdFJ4bVVHVExENDNKVHoxaVdiVnViQT0=
owner_type | string | 否 | 分类的所属类型<br>**示例值**：user<br>**可选值有**：<br>- user：员工<br>- department：部门<br>**默认值**：`user`

## 响应

### 响应体

名称 | 类型 | 描述
---|---|---
code | int | 错误码，非 0 表示失败
msg | string | 错误描述
data | \- | \-
has_more | boolean | 是否还有更多项
page_token | string | 分页标记，当 has_more 为 true 时，会同时返回新的 page_token，否则不返回 page_token
items | category\[\] | 分类列表
id | string | 分类 ID
create_time | string | 分类的创建时间，毫秒级时间戳
update_time | string | 分类的更新时间，毫秒级时间戳
category_type | string | 分类类型<br>**可选值有**：<br>- person：个人<br>- team：团队
enabled | boolean | 是否启用
color | string | 颜色<br>**可选值有**：<br>- blue：蓝色<br>- purple：紫色<br>- wathet：浅蓝色<br>- turquoise：蓝绿色<br>- indigo：靛青<br>- orange：橙色
name | category_name | 名称
zh | string | 中文名
en | string | 英文名
ja | string | 日文名
