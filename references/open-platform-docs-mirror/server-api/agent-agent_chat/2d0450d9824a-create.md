# 发起智能体对话

异步发起一轮智能体对话，提交用户消息后立即返回对话ID，触发智能体在后台运行。
**注意事项**：需通过[获取对话结果](/uAjLw4CM/ukTMukTMukTM/aily-v1/agent-agent_chat/get)接口轮询运行状态与回复。

## 请求

基本 | &nbsp;
---|---
HTTP URL | https://open.feishu.cn/open-apis/aily/v1/agents/:agent_id/chats
HTTP Method | POST
接口频率限制 | [10 次/秒](https://open.feishu.cn/document/ukTMukTMukTM/uUzN04SN3QjL1cDN)
支持的应用类型 | Custom App
权限要求<br>**调用该 API 所需的权限。开启其中任意一项权限即可调用** | 发起对话与管理会话(aily:agent_chat:write)

### 请求头

名称 | 类型 | 必填 | 描述
---|---|---|---
Authorization | string | 是 | `tenant_access_token`<br>或<br>`user_access_token`<br>**值格式**："Bearer `access_token`"<br>**示例值**："Bearer u-7f1bcd13fc57d46bac21793a18e560"<br>[了解更多：如何选择与获取 access token](https://open.feishu.cn/document/uAjLw4CM/ugTN1YjL4UTN24CO1UjN/trouble-shooting/how-to-choose-which-type-of-token-to-use)
Content-Type | string | 是 | **固定值**："application/json; charset=utf-8"

### 路径参数

名称 | 类型 | 描述
---|---|---
agent_id | string | 智能体id，通过智能体后台详情的地址栏中获取<br>**示例值**："agent_4k4ue29hpwrx2"<br>**数据校验规则**：<br>- 长度范围：`0` ～ `65` 字符

### 请求体

名称 | 类型 | 必填 | 描述
---|---|---|---
user_message | agent_user_message | 是 | 用户请求对话内容，必填参数
content | agent_user_message_content\[\] | 是 | 用户消息，元素最多100
type | string | 是 | 内容类型<br>- 目前仅支持text<br>**示例值**："text"<br>**数据校验规则**：<br>- 长度范围：`0` ～ `64` 字符
text | string | 是 | 内容<br>**示例值**："你好！有什么可以帮您"<br>**数据校验规则**：<br>- 长度范围：`0` ～ `10000` 字符
agent_attachment_ids | string\[\] | 否 | 附件id，通过调用[上传附件](https://open.larkoffice.com/document/uAjLw4CM/ukTMukTMukTM/aily-v1/agent-agent_attachment/create)接口获取<br>**示例值**：["3d058789-6952-4697-bf9c-1add1ebc206e"]<br>**数据校验规则**：<br>- 长度范围：`0` ～ `8`

### 请求体示例
```json
{
    "user_message": {
        "content": [
            {
                "type": "text",
                "text": "你好！有什么可以帮您"
            }
        ],
        "agent_attachment_ids": [
            "3d058789-6952-4697-bf9c-1add1ebc206e"
        ]
    }
}
```

## 响应

### 响应体

名称 | 类型 | 描述
---|---|---
code | int | 错误码，非 0 表示失败
msg | string | 错误描述
data | \- | \-
agent_chat_id | string | 对话ID

### 响应体示例
```json
{
    "code": 0,
    "msg": "success",
    "data": {
        "agent_chat_id": "7640186506971926032"
    }
}
```

### 错误码

HTTP状态码 | 错误码 | 描述 | 排查建议
---|---|---|---
400 | 2700001 | param is invalid | 检查参数是否正确
