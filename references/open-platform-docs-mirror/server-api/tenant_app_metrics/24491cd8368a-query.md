# 获取应用运营数据

获取 aPaaS 应用活跃数据、存储或运行资源用量数据。

## 请求

基本 | &nbsp;
---|---
HTTP URL | https://open.feishu.cn/open-apis/apaas/v1/tenant_app_metrics/query
HTTP Method | POST
接口频率限制 | [10 次/秒](https://open.feishu.cn/document/ukTMukTMukTM/uUzN04SN3QjL1cDN)
支持的应用类型 | Custom App、Store App
权限要求<br>**调用该 API 所需的权限。开启其中任意一项权限即可调用** | 获取应用运营数据(app_engine:tenant_app_metrics:read)

### 请求头

名称 | 类型 | 必填 | 描述
---|---|---|---
Authorization | string | 是 | `user_access_token`<br>**值格式**："Bearer `access_token`"<br>**示例值**："Bearer u-7f1bcd13fc57d46bac21793a18e560"<br>[了解更多：如何选择与获取 access token](https://open.feishu.cn/document/uAjLw4CM/ugTN1YjL4UTN24CO1UjN/trouble-shooting/how-to-choose-which-type-of-token-to-use)
Content-Type | string | 是 | **固定值**："application/json; charset=utf-8"

### 请求体

名称 | 类型 | 必填 | 描述
---|---|---|---
namespaces | string\[\] | 是 | aPaaS 应用 namespace 列表，单次请求最多查询 20 个 namespace。namespace 可以通过 aPaaS 平台-开放中心-应用管理页面获取<br>**示例值**：["package_a__c"]
date | string | 是 | 查询日期，支持查询最近一年至前一日数据。日期格式为 YYYY-MM-DD<br>**示例值**："2006-01-02"

### 请求体示例
```json
{
    "namespaces": [
        "package_a__c"
    ],
    "date": "2006-01-02"
}
```

## 响应

### 响应体

名称 | 类型 | 描述
---|---|---
code | int | 错误码，非 0 表示失败
msg | string | 错误描述
data | \- | \-
metrics | tenant_app_metrics\[\] | 查询应用的指标详情列表
namespace | string | aPaaS 应用 namespace
data | string | 指定应用的指标详情，对 []map[string]int64 数组进行序列化后得到的 json 字符串。数组每个元素包含名称（Name）、数值类型（ValueType）、数值（Value）三个字段。<br>Name枚举值：<br>active_user_1dcnt（近1日应用用户活跃数）、<br>active_user_7dcnt（近7日应用用户活跃数）、<br>active_user_30dcnt（近30日应用用户活跃数）、<br>function_run_1dcnt（近1日函数运行次数，含线上环境与开发环境）。<br>ValueType枚举值：amount（数量）。<br>json格式：[{"Name":"active_user_1dcnt","ValueType":"amount","Value":100},<br>{"Name":"active_user_7dcnt","ValueType":"amount","Value":200},<br>...<br>]

### 响应体示例
```json
{
    "code": 0,
    "msg": "success",
    "data": {
        "metrics": [
            {
                "namespace": "package_a__c",
                "data": "[{\"Name\":\"active_user_30dcnt\",\"ValueType\":\"amount\",\"Value\":6000},{\"Name\":\"active_user_7dcnt\",\"ValueType\":\"amount\",\"Value\":1400},{\"Name\":\"active_user_1dcnt\",\"ValueType\":\"amount\",\"Value\":200},{\"Name\":\"function_run_1dcnt\",\"ValueType\":\"amount\",\"Value\":400}]"
            }
        ]
    }
}
```

### 错误码

HTTP状态码 | 错误码 | 描述 | 排查建议
---|---|---|---
404 | 2320003 | tenant does not exist | 请检查传入的租户ID是否正确，或联系租户管理员确认租户状态是否正常
404 | 2320004 | user does not exist | 请检查传入的用户ID是否正确
403 | 2320005 | insufficient permissions | 请确认用户是否拥有该接口所需的权限（【获取应用运营数据】权限），如何申请权限可参考相关权限文档
400 | 2320006 | bad request or invalid parameter | 请自查请求方法、请求头（如Content-Type）、请求体参数及请求数据格式等是否正确
