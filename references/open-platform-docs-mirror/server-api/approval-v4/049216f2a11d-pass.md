# 同意审批任务

对于单个审批任务进行同意操作。同意后审批流程会流转到下一个审批人。

## 请求

基本 | &nbsp;
---|---
HTTP URL | https://open.feishu.cn/open-apis/approval/v4/tasks/pass
HTTP Method | POST
接口频率限制 | [100 次/分钟](https://open.feishu.cn/document/ukTMukTMukTM/uUzN04SN3QjL1cDN)
支持的应用类型 | Custom App、Store App
权限要求<br>**调用该 API 所需的权限。开启其中任意一项权限即可调用** | 处理审批任务(approval:task:write)

### 请求头

名称 | 类型 | 必填 | 描述
---|---|---|---
Authorization | string | 是 | `user_access_token`<br>**值格式**："Bearer `access_token`"<br>**示例值**："Bearer u-7f1bcd13fc57d46bac21793a18e560"<br>[了解更多：如何选择与获取 access token](https://open.feishu.cn/document/uAjLw4CM/ugTN1YjL4UTN24CO1UjN/trouble-shooting/how-to-choose-which-type-of-token-to-use)
Content-Type | string | 是 | **固定值**："application/json; charset=utf-8"

### 请求体

名称 | 类型 | 必填 | 描述
---|---|---|---
instance_code | string | 是 | 审批实例 Code，可通过审批列表相关接口获取<br>**示例值**："81D31358-93AF-92D6-7425-01A5D67C4E71"
task_id | string | 是 | 审批任务 ID，可通过审批列表相关接口获取<br>**示例值**："123456789"
form | string | 否 | 表单数据，默认不需要传，仅有该节点需要录入的表单字段才需要传入<br>**示例值**："[{"id":"user_name", "type": "input", "value":"ou_123avcssa111"}]"
comment | string | 否 | 审批意见<br>**示例值**："同意"

### 请求体示例
```json
{"instance_code":"81D31358-93AF-92D6-7425-01A5D67C4E71",
"task_id":"123456789",
"form":"[{"id":"user_name", "type": "input", "value":"ou_123avcssa111"}]",
"comment":"同意"}
```
