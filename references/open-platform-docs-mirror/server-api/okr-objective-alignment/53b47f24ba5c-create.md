# 创建目标对齐关系

为指定目标与另一个目标创建对齐关系，在 OKR 结构中建立层级对齐。

## 请求

基本 | &nbsp;
---|---
HTTP URL | https://open.feishu.cn/open-apis/okr/v2/objectives/:objective_id/alignments
HTTP Method | POST
接口频率限制 | [100 次/分钟](https://open.feishu.cn/document/ukTMukTMukTM/uUzN04SN3QjL1cDN)
支持的应用类型 | Custom App
权限要求<br>**调用该 API 所需的权限。开启其中任意一项权限即可调用** | 更新 OKR 内容(okr:okr.content:writeonly)

### 请求头

名称 | 类型 | 必填 | 描述
---|---|---|---
Authorization | string | 是 | `tenant_access_token`<br>或<br>`user_access_token`<br>**值格式**："Bearer `access_token`"<br>**示例值**："Bearer u-7f1bcd13fc57d46bac21793a18e560"<br>[了解更多：如何选择与获取 access token](https://open.feishu.cn/document/uAjLw4CM/ugTN1YjL4UTN24CO1UjN/trouble-shooting/how-to-choose-which-type-of-token-to-use)
Content-Type | string | 是 | **固定值**："application/json; charset=utf-8"

### 路径参数

名称 | 类型 | 描述
---|---|---
objective_id | string | 目标 ID。可通过 [获取用户 OKR 周期内的目标] 接口获取。<br>**示例值**："7342342398472398473"<br>**数据校验规则**：<br>- 长度范围：`1` ～ `20` 字符

### 请求体

名称 | 类型 | 必填 | 描述
---|---|---|---
to_entity_type | int | 是 | 被对齐实体类型<br>**示例值**：2<br>**可选值有**：<br>- 2：目标
to_entity_id | string | 是 | 被对齐实体(目标) ID，可通过“获取用户 OKR 周期内的目标”接口获取。<br>**示例值**："7342342398472398473"<br>**数据校验规则**：<br>- 长度范围：`1` ～ `20` 字符

### 请求体示例
```json
{
    "to_entity_type": 2,
    "to_entity_id": "7342342398472398473"
}
```

## 响应

### 响应体

名称 | 类型 | 描述
---|---|---

msg | string | 错误描述
data | \- | \-
