# 简介

本教程介绍如何使用飞书开放平台通讯录开放能力，完成对部门的增删改查操作，并监听员工的入职、离职事件。通过本教程你可以快速了解如何使用 OpenAPI 操作通讯录，以及处理员工入职、离职事件。
