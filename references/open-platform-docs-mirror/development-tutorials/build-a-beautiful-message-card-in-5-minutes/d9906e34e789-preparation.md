# 准备工作

在开始本教程之前，你需要完成以下准备工作：
* [拥有](https://www.feishu.cn/hc/zh-CN/articles/360043741453)或[加入](https://www.feishu.cn/hc/zh-CN/articles/360043496893)飞书企业。
