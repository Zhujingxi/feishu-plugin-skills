# 切换文档

你可以通过接口切换组件内文档地址。

## 使用方式
```js
myComponent.replace(src);
```

## 参数
|属性|	类型|	是否必须	|说明|
| ---|----- | -------|------ |
|src|	String	|是|	文档地址
