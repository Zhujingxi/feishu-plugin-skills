# 打开第三方日历账户管理页 
从飞书 3.47.0 版本开始支持。

## 使用场景
打开第三方日历账户管理页，方便用户添加或导入谷歌/Exchange 日历。移动端打开页面，PC端打开弹层。

## 协议
[https://applink.feishu.cn/client/calendar/account](https://applink.feishu.cn/client/calendar/account)

##  参数
无
