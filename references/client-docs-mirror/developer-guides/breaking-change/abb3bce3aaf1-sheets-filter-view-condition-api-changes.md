# 表格筛选视图筛选条件相关接口变更
### 变更事项

为了进一步优化表格性能，筛选视图筛选条件 `condition` 中的筛选类型 `filter_type`参数将于**2026年6月15日**起不再支持隐藏值筛选 `hiddenValue`，新增支持多值筛选 `multiValue`。

多值筛选指筛选特定的值。当筛选类型为多值时：
* `compare_type`：不填
* `expected`：要展示的值。数组类型，数组长度需大于 0 且小于等于筛选范围的行数。每个值的字符长度不得超过 50,000 个。

```json
{
  "condition": {
    "filter_type": "multiValue",
    "expected": ["100", "200", "300"]  // 筛选出数据为 100，200, 和 300 的单元格
  }
}
```

影响接口：
- [创建筛选条件](https://open.feishu.cn/document/ukTMukTMukTM/uUDN04SN0QjL1QDN/sheets-v3/spreadsheet-sheet-filter_view-condition/create)
- [更新筛选条件](https://open.feishu.cn/document/ukTMukTMukTM/uUDN04SN0QjL1QDN/sheets-v3/spreadsheet-sheet-filter_view-condition/update)
- [查询筛选条件](https://open.feishu.cn/document/ukTMukTMukTM/uUDN04SN0QjL1QDN/sheets-v3/spreadsheet-sheet-filter_view-condition/query)
- [获取筛选条件](https://open.feishu.cn/document/ukTMukTMukTM/uUDN04SN0QjL1QDN/sheets-v3/spreadsheet-sheet-filter_view-condition/get)

### 潜在影响

**2026年6月15日**起，创建/更新筛选条件写入隐藏值筛选 `hiddenValue`，API 将会返回一个错误。

### 解决方案

不再使用隐藏值筛选 hiddenValue，转换为使用多值筛选 multiValue。
