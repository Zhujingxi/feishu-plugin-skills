# 消息与群组 API 支持使用 user_access_token 获取外部会话数据

### 变更事项

为提供更完善的消息与群组开放能力，**2026 年 6 月 8 日起，应用使用本企业成员的 `user_access_token` 调用相关 API 时，将支持获取外部会话（含单聊与群组）中的消息与群组数据，不再返回无外部会话权限等报错。**

<br>
**变更之前：** 在校验 `user_access_token` 是否有权限获取外部会话的数据时，部分接口会结合应用的 **对外共享状态、关联组织共享状态** 进行判断，如果不满足条件则会返回无外部会话权限错误码

**变更之后：** 在校验 `user_access_token` 是否有权限获取外部会话的数据时，不再判断应用的 **对外共享状态、关联组织共享状态**

<br>
具体受影响接口包括：

1. [获取指定消息的内容](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/message/get)
2. [获取群内 Pin 消息](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/pin/list)
3. [获取消息表情回复](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/message-reaction/list)
4. [获取消息中的资源文件](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/message-resource/get)
5. [获取群信息](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/chat/get)
6. [判断用户或机器人是否在群里](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/chat-members/is_in_chat)
7. [获取群成员列表](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/chat-members/get)

> **是否跟随客户端版本**：不跟版<br>
> **预计生效时间**：2026 年 6 月 8 日<br>

### 潜在影响

本次变更预计不会对现有应用造成阻塞性影响，但需关注应用获取到的消息可能包含**外部租户**参与的会话内容
