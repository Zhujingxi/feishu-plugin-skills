# 开发者工具

**1. 旧版的 IDE 工具和 VSCode 插件还能下载、使用吗？**

答：飞书小程序 IDE 和 VSCode 插件工具已不再更新，建议使用[新版飞书开发者工具](https://open.feishu.cn/document/uYjL24iN/ucDOzYjL3gzM24yN4MjN)

<br>
**2. 我上传、预览小程序时，为什么显示没有权限呢？**

答：目前仅应用的**创建者** 、**管理员**和**开发者**可以上传/预览小程序
