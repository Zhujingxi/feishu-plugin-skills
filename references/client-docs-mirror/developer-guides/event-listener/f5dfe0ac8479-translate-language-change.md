# 文档翻译变化
## 示例
```js
// npm
myComponent.register(DocComponentEvent.TRANSLATE_CHANGE, function(language) {
  // ...
});

// sdk
myComponent.register('TRANSLATE_CHANGE', function(language) {
  // ...
});
```

## 返回
|属性|	类型|	说明|
| ---|----- | ------- | 
|language|	String |当前翻译语言|
