# 文档滚动时

## 示例
```js
// npm
myComponent.register(DocComponentEvent.DOC_EDITOR_SCROLL, function(scrollTop) {
});

// sdk
myComponent.register('DOC_EDITOR_SCROLL', function(scrollTop) {
});
```

## 返回
|属性|	类型|	说明|
| ---|----- | ------- | 
|scrollTop|	Number |滚动距离|
