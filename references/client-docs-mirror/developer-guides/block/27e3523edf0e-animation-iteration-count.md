# animation-iteration-count

## 介绍

用于设置播放动画的次数。

## 语法

```css
/* keyword */

animation-iteration-count: infinite;

/* number */

animation-iteration-count: 5;
```

### 取值

-   `infinite`

无限循环播放动画。

-   `<number>`

**默认值。** 动画播放的次数；默认值为`1`，不可为负值。

## 标准化语法

```css
animation-iteration-count: <number> | infinite
```
