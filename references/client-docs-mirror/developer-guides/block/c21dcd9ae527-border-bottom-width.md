# border-bottom-width

## 介绍

用于设置下边框的宽度

## 语法

```css
/* 下边框是细边框 */

border-bottom-width: thin;

/* 下边框是中等边框 */

border-bottom-width: medium;

/* 下边框是粗边框 */

border-bottom-width: thick;

/* 下边框是10px */

border-bottom-width: 10px;
```

### 取值

-   `thin`：细边框。

-   `medium`： 中等边框。

-   `thick`：粗边框。

-   `<length>`：以具体的尺寸定义边框宽度

## 标准化语法

```css
border-bottom-width: thin | medium | thick | <length>
```
