# CanvasContext.setTextBaseline(string align)

设置字体的对齐基线

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **✓** | V7.35.0+ | 预览
网页应用 | **X** | **X** | **X** | **X** | /

## 输入

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
align | string | 是 | &nbsp; | 对齐方式<br>**可选值**：<br>- `top` 字体顶部基线<br>- `hanging` 字体悬停基线<br>- `middle` 字体中部基线<br>- `alphabetic` 字体默认基线<br>- `ideographic` 实际渲染字体基线，通常用于表意文字<br>- `bottom` 基线为字体底线

## 输出

无

## 示例代码

```javascript
const textBaselineList = ["top", "middle", "bottom", "normal"];

ctx.strokeStyle = "red";
ctx.moveTo(5, 75);
ctx.lineTo(295, 75);
ctx.stroke();
ctx.font = "20px sans-serif";

for (let i = 0; i < textBaselineList.length; ++i) {
  ctx.setTextBaseline(textBaselineList[i]);
  ctx.fillText(textBaselineList[i], 5 + 70 * i, 75);
}

ctx.draw();
```
