# CanvasContext.fillText(string text, number x, number y, number maxWidth)

填充文字

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **✓** | V7.35.0+ | 预览
网页应用 | **X** | **X** | **X** | **X** | /

## 输入

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
text | string | 是 | &nbsp; | 文字
x | number | 是 | &nbsp; | x 坐标
y | number | 是 | &nbsp; | y 坐标
maxWidth | number | 否 | &nbsp; | 文字最大宽度

## 输出

无

## 示例代码

```javascript
const ctx = tt.createCanvasContext(canvasId);

ctx.setFontSize(20);
ctx.fillText("Hello Block!", 20, 20);

ctx.draw();
```
