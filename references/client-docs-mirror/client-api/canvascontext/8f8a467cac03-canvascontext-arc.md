# CanvasContext.arc(number x, number y, number radius, number startAngle, number endAngle, string counterclockwise)

添加圆弧到当前路径中

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **✓** | V7.35.0+ | 预览
网页应用 | **X** | **X** | **X** | **X** | /

## 输入

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
x | number | 是 | &nbsp; | 圆心 x 坐标
y | number | 是 | &nbsp; | 圆心 y 坐标
radius | number | 是 | &nbsp; | 半径
startAngle | number | 是 | &nbsp; | 圆开始弧度
endAngle | number | 是 | &nbsp; | 圆结束弧度
counterclockwise | string | 否 | false | 是否是逆时针计算

## 输出

无

## 示例代码

```javascript
ctx.arc(100, 100, 60, 0, 1.5 * Math.PI, false);
ctx.fill();

ctx.draw();
```
