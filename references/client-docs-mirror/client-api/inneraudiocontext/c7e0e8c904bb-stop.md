# InnerAudioContext.stop()

停止播放音频

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **✕** | V7.43.0+ | 预览
网页应用 | **✕** | **✕** | **✕** | **✕** | /

## 输入
无

## 输出
无

## 示例代码

```js
const innerAudioContext = tt.createInnerAudioContext();
innerAudioContext.autoplay = true;
innerAudioContext.src = 'https://someaudiourl';
innerAudioContext.play();
innerAudioContext.stop();
```
