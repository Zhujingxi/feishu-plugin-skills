# NFCAdapter.getNfcA()

获取NfcA实例，实例支持NFC-A (ISO 14443-3A)标准的读写

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | V3.38.0+ | V5.25.0+ | **X** | V7.35.0+ | 预览
网页应用 | V3.44.0+ | V5.25.0+ | **X** | V7.35.0+ | 预览

## 输入
无

## 示例代码

**注意事项**：这里仅示例当前 API 使用，实际流程请参考：[接入 NFC](https://open.feishu.cn/document/uYjL24iN/ugTN4YjL4UDO24CO1gjN)
