# getChatInfo(Object object)

获取某个会话的信息
**注意事项**：小程序调用该接口前，需要确保已经调用 [requestAccess](https://open.feishu.cn/document/uYjL24iN/uUzMuUzMuUzM/requestaccess) ( 如需兼容 飞书V6.9.0 以下版本，可使用 [login](https://open.feishu.cn/document/uYjL24iN/uYzMuYzMuYzM) )

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | V3.10.0+ | V3.10.0+ | V3.10.0+ | V7.35.0+ | 预览
网页应用 | V3.44.0+ | V3.44.0+ | V3.44.0+ | V7.35.0+ | 预览

## 输入

继承[标准对象输入](https://open.feishu.cn/document/uYjL24iN/ukzNy4SO3IjL5cjM)，扩展属性描述：

名称 | 数据类型 | 必填 | 默认值 | 描述
---|---|---|---|---
openChatId | string | 是 | &nbsp; | 获取会话信息的会话[open_chat_id](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/im-v1/chat-id-description)<br>**示例值**：oc_1965ed81fc91d3b73d68c4ca4cfc110a
chatType | number | 是 | &nbsp; | 会话的类型<br>**示例值**：0<br>**可选值**：<br>- `0`：单聊<br>- `1`：群聊
userType | number | 否 | &nbsp; | 单聊用户类型<br>**示例值**：0<br>**可选值**：<br>- `0`：用户<br>- `1`：bot<br>**注意事项**：chatType为0时，必须传该参数

## 输出

`success`返回对象的扩展属性：

名称 | 数据类型 | 描述
---|---|---
i18nNames | object | 国际化会话名(可能为空)<br>**字段权限要求**：<br>读取群信息(im:chat.group_info:readonly)
&emsp;<br>∟<br>&nbsp;<br>zh_cn | string | 中文名，可能为空
&emsp;<br>∟<br>&nbsp;<br>en_us | string | 英文名，可能为空
&emsp;<br>∟<br>&nbsp;<br>ja_jp | string | 日文名，可能为空
name | string | 会话名称<br>**字段权限要求**：<br>读取群信息(im:chat.group_info:readonly)
avatarUrls | string[] | 会话的头像url数组，包含多种图片分辨率<br>**字段权限要求**：<br>读取群信息(im:chat.group_info:readonly)
atCount | number | 被at数量<br>**字段权限要求**：<br>读取群信息(im:chat.group_info:readonly)<br>**注意事项**：飞书[V3.12.0](https://open.feishu.cn/document/uYjL24iN/uAjMuAjMuAjM/version-compatibility)及以上版本支持
badge | number | 未读消息数<br>**字段权限要求**：<br>读取群信息(im:chat.group_info:readonly)

## 示例代码

```js
tt.getChatInfo(
  {
    openChatId: 'oc_1965ed81fc91d3b73d68c4ca4cfc110a',
    chatType:  0,
    userType:  0,
    success (res) {
        console.log(JSON.stringify(res));
    },
    fail (res) {
        console.log(`getChatInfo fail:${JSON.stringify(res)}`);
    }
  }
)
```
