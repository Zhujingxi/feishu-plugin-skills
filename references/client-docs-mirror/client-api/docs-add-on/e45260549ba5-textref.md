# TextRef
文本的引用对象。

名称 | 数据类型 | 是否必填 | 描述
---|---|---|---
blockRef | [BlockRef](https://open.feishu.cn/document/uAjLw4CM/uYjL24iN/docs-add-on/05-api-doc/basic-data-reference---base/BlockRef) | 是 | 对应的 Block 引用
range | [number, number] | 是 | 文本范围，假设数组为 [start, end]，则：<br>- start > end 时，是非法值，会降级为 [start, start]<br>- start = end 时，表示文本位置<br>- start < end 时，表示文本范围<br>start 和 end 超出 Block 的文本内容时，则设置为文本内容的最后位置。
