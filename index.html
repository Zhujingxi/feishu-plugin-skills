Feishu/Lark Plugin Development Skill

Install the full feishu-plugin-development Hermes skill, including all reference files.

curl -fsSL https://YOUR-VERCEL-DOMAIN.vercel.app/api/install | bash

If this page is deployed on Vercel, replace YOUR-VERCEL-DOMAIN.vercel.app with the current domain.

Install into a named Hermes profile:

HERMES_PROFILE=coder curl -fsSL https://YOUR-VERCEL-DOMAIN.vercel.app/api/install | bash
